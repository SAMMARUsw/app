## 🔑 서비스 주요 기능
**1. 공모전 팀원 모집**

    공모전 팀원을 모집하는 글을 작성할 수 있습니다.

**2. 공모전 팀원 지원**

    공모전 팀원을 모집하는 팀에 지원서를 제출할 수 있습니다. 제출한 지원서는 해당 팀장에게 전달되며, 수락 또는 거절을 할 수 있습니다.

**3. 실시간 공모전 정보**

    진행중인 공모전을 실시간으로 조회할 수 있습니다.
